# 🧠📱 OpenDriver VR Mobile Launcher

**Turn your old phone into a futuristic standalone VR headset. No cables. No limits. Just pure imagination.**

---

## 🚀 About the Project

**OpenDriver VR Launcher** is an Android application that transforms your old smartphone into a **standalone VR headset experience**, inspired by modern devices like Quest-style systems… but built from scratch, by the community.

Think of it as:
> *“Gear VR… if it time-traveled to 2026 and got an upgrade.”*

This launcher provides a full VR environment where your phone becomes the brain of your headset — running apps, handling input, and immersing you in virtual worlds without needing a PC.

---

## 🧩 Features

- 🕶️ Standalone VR experience on your phone  
- ✋ Built-in **hand tracking**  
- 🎮 VR app launcher environment (Quest-like UI)  
- 🔓 Fully **open-source**  
- 🧱 Built in **Unity**  
- 🔧 Easy to modify, extend, and build your own features  
- 📦 Ready-to-use APK available in **Releases**  
- 🧠 Designed for experimentation, creativity, and hacking  

---

## 🛠️ Installation

### Option 1: Download APK (Easy)
1. Go to the **Releases** tab
2. Download the latest `.apk`
3. Install it on your Android device
4. Launch and enter VR 🌌

### Option 2: Build it Yourself (Recommended for devs)
1. Clone the repository
2. Open the project in **Unity**
3. Build for Android
4. Install on your device

Everything you need is included — no hidden magic.
